  • 넥센타이어 렌탈료 자동이체 건이 여러 건인 경우에도 합산하여 할인한도 내에서 할인 적용
예시 전월실적 100만원 이상 시
  • 넥센타이어 렌탈 요금 2만 5천원 1건인 경우
    → 2만원 할인
  • 넥센타이어 렌탈 요금 1만원 2건인 경우
    → 각 1만원/1만원 (총 2만원) 할인
  • 넥센타이어 렌탈 요금 1만원 1건인 경우
    → 1만원 할인
